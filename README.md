# Padel Tracker

App for å spore padel-kamper og statistikk for en gruppe spillere.

## Funksjoner

- Legg til og fjern spillere
- Registrer kampresultater (2 vs 2)
- Statistikk per spiller: kamper, seire, tap, sett vunnet/tapt
- Kamphistorikk med dato og resultat
- Data lagres lokalt i nettleseren (localStorage)

## Kom i gang

```bash
npm install
npm run dev
```

Besøk `http://localhost:3000`

## Teknologi

- [Next.js](https://nextjs.org/) 14
- [React](https://react.dev/) 18
- [Tailwind CSS](https://tailwindcss.com/)
- [Lucide React](https://lucide.dev/) (ikoner)
